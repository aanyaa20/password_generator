# 🔐 React Password Generator

A simple and customizable password generator built with **React** and **Tailwind CSS**. Generate strong random passwords with adjustable length, optional numbers, and special characters. Includes a copy-to-clipboard feature.

---

## ✨ Features

- Adjustable password length (6 to 100)
- Toggle inclusion of:
  - Numbers (0-9)
  - Special characters (!@#$%^&*...)
- Copy password to clipboard with a single click
- Fully responsive and clean UI

---

## 🛠️ Tech Stack

- React (Hooks: `useState`, `useEffect`, `useRef`, `useCallback`)
- Tailwind CSS
- JavaScript (ES6+)
